# 中国国家地理标志产品技术报告

## 1. 项目概述

### 1.1 项目背景

中国国家地理标志产品是指产自特定地域，所具有的质量、声誉或其他特性本质上取决于该产地的自然因素和人文因素，经审核批准以地理名称进行命名的产品。地理标志产品不仅是地方特色经济的重要组成部分，也是中华传统文化的重要载体。

随着地理标志保护制度的不断完善，官方发布的地理标志保护产品批准公告数量逐年增加。然而，这些公告信息以PDF格式分散发布，缺乏系统性的数据整理和分析，难以充分挖掘其中的价值。

### 1.2 项目目标

本项目旨在构建一个全面、系统的中国国家地理标志产品数据库，并基于该数据库开展分析和可视化，具体目标包括：

1. 系统性收集官方发布的地理标志保护产品批准公告
2. 从公告中提取产品名称、产地、感官特征等关键信息
3. 对提取的数据进行规范化处理和自然语言分析
4. 开展地理分布分析，揭示地理标志产品的区域分布特征
5. 进行感官文本挖掘，探索产品描述的语言规律和地域特色

### 1.3 项目意义

本项目的实施具有以下重要意义：

1. **数据价值**：构建了全网最新、最全、最适用于数据分析的中国国家地理标志产品数据库
2. **研究价值**：为地理标志产品的区域分布、特色形成机制等研究提供数据支持
3. **应用价值**：为地方政府、企业和消费者提供地理标志产品的全面信息
4. **技术价值**：展示了从非结构化PDF数据到结构化分析的完整技术流程

## 2. 技术架构

本项目采用分层架构设计，主要包括数据采集部分、数据处理部分、分析部分和展示部分四个部分：

| 层级 | 功能描述 | 主要组件 |
|------|---------|----------|
| 数据采集层 | 从官方网站下载地理标志保护产品批准公告PDF文件 | Selenium |
| 数据处理层 | 从PDF中提取信息、文本润色、数据预处理、自然语言处理 | AI大模型（Qwen3-Max）、HanLP |
| 分析层 | 地理分布分析、感官文本挖掘 | Pandas、Matplotlib、Pyecharts |
| 展示层 | 生成可视化图表、词云等 | Matplotlib、Pyecharts、WordCloud |

## 3. 数据处理流程

### 3.1 数据来源

本项目的数据来源于中国国家地理标志产品检索网站（https://ggfw.cnipa.gov.cn/dlbzsq/dbQuery），具体包括：

1. **原质监总局地理标志保护产品批准公告**（2005-2018年）
2. **国家知识产权局地理标志保护产品批准公告**（2019年至今）

原始公告文件以PDF格式存储在 `data/origin/` 目录下。

### 3.2 数据处理步骤

数据处理流程包括以下五个主要步骤：

1. **下载公告**
   - 从官方网站爬取地理标志保护产品批准公告PDF文件
   - 按年份和公告编号组织文件存储
2. **提取信息**
   - 使用AI大模型（Qwen3-Max）从PDF文件中提取文本内容
   - 将提取的信息存储为JSON格式
3. **文本润色**
   - 对提取的文本进行润色处理（Qwen3-Max）
   - 修正识别错误和格式问题
4. **数据预处理**
   - 清洗数据，去除重复记录
5. **自然语言处理**
   - 使用HanLP进行中文分词
   - 进行词性标注和命名实体识别

### 3.3 数据结构

经过处理后的数据存储为两种格式：

1. **基础数据**（`data/products.json`）
   ```json
   {
       "product_name": "产品名称",
       "province": "省份",
       "city": "城市",
       "districts": ["区县1", "区县2", ...],
       "sensory_characteristics": "感官特征描述"
   }
   ```

2. **分词后数据**（`data/tokenized_products.json`）
   ```json
   {
     "product_name": "产品名称",
     "tokenized_product_name": {
         "tok/fine": [],
         "pos/ctb": [],
         "ner/msra": [],
         "con": []
     },
     "province": "省份",
     "city": "城市",
     "districts": ["区县1", "区县2", ...],
     "sensory_characteristics": "感官特征描述",
     "tokenized_sensory_characteristics": {
         "tok/fine": [],
         "pos/ctb": [],
         "ner/msra": [],
         "con": []
     }
   }
   ```

## 4. 核心功能模块

### 4.1 数据采集模块

**功能描述**：从官方网站自动下载地理标志保护产品批准公告PDF文件。

**实现细节**：
- 使用Selenium库模拟浏览器操作
- 解析网页结构，提取PDF下载链接
- 按年份和公告编号组织文件存储
- 记录下载日志，确保数据完整性

**关键技术点**：
- Selenium库的使用
- 批量文件下载
- 错误处理与重试机制

### 4.2 信息提取模块

**功能描述**：从PDF公告中提取产品名称、产地、感官特征等关键信息。

**实现细节**：
- 使用AI大模型（Qwen3-Max）解析PDF文件
- 将提取的信息转换为结构化JSON格式

**关键技术点**：

- 提示词构建
- 信息结构化

### 4.3 自然语言处理模块

**功能描述**：对产品数据进行中文分词、词性标注和命名实体识别。

**实现细节**：
- 使用HanLP进行中文分词和词性标注
- 识别产品名称、地名等命名实体
- 分析产品描述的句法结构
- 生成分词后的结构化数据

**关键技术点**：
- 中文分词算法
- 词性标注
- 命名实体识别

### 4.4 地理分布分析模块

**功能描述**：分析地理标志产品的区域分布特征，生成可视化图表。

**实现细节**：

- 使用Pyecharts生成地理分布热力图
- 分析产品分布的区域差异
- 生成省份和城市级别的产品词云图

**关键技术点**：
- 地理数据可视化
- 热力图生成
- 词云图生成

### 4.5 感官文本挖掘模块

**功能描述**：分析产品感官特征描述的语言规律和地域特色。

**实现细节**：
- 提取产品描述中的形容词和感官词汇
- 分析同类产品的感官形容词共性
- 对比南北地区产品的感官描述差异
- 绘制各省份最热门的颜色分布图

**关键技术点**：
- 文本特征提取
- 情感分析
- 地域差异对比

## 5. 技术实现细节

### 5.1 信息结构化技术

从非结构化文本中提取结构化信息是本项目的核心技术难点之一。由于不同年份、不同来源的PDF文件格式差异较大，我们采用了以下策略：

1. **AI大模型**：使用AI大模型（Qwen3-MAX）将提取范式转换为生成范式。
2. **文件处理**：由于大模型的上下文长度限制，大模型每次只能处理50个左右的产品，需要对PDF文件进行精细的分割处理。
3. **提示词优化**：根据实际提取效果不断调整和优化提示词
4. **多轮验证**：通过多轮验证确保提取信息的一致性
5. **人工审核**：对复杂案例进行人工审核和修正

### 5.2 自然语言处理技术

本项目使用HanLP进行中文自然语言处理，为后续分析提供基础，主要应用了以下技术：

1. **分词技术**：使用细粒度分词模型，提高分词准确率
2. **词性标注**：对分词结果进行词性标注，为后续分析提供基础
3. **命名实体识别**：识别产品名称、地名等命名实体

### 5.3 数据可视化技术

为了直观展示分析结果，我们采用了多种数据可视化技术：

1. **地理热力图**：使用Pyecharts生成交互式地理热力图，展示产品分布
2. **词云图**：使用WordCloud生成词云图，展示产品多样性
3. **柱状图**：使用Matplotlib生成柱状图，展示数量对比
4. **颜色图谱**：设计颜色图谱，展示各省份产品的颜色特征

## 6. 实验结果与分析

### 6.1 数据规模与质量

**数据规模**：
- 收集了2005年至今的地理标志保护产品批准公告
- 构建了包含4400+个地理标志产品的数据库
- 覆盖31个省级行政区、300+个地级市

**数据质量**：
- 产品提取准确率达到95%以上
- 文本规范化处理效果良好，一致性高

### 6.2 地理分布分析结果

**区域分布特征**：
- 地理标志产品数量呈现明显的区域差异
- 东部、中部地区产品数量较多，西部地区相对较少
- 人文底蕴浓厚的省份，如四川省、山东省、湖北省、广西壮族自治区、广东省等产品数量领先

**热点区域**：
- 省级热点：四川省、山东省、湖北省、广西壮族自治区、广东省
- 市级热点：成都市、重庆市、黄冈市、青岛市、十堰市

**分布规律**：

- 地理标志产品分布与当地自然资源禀赋密切相关
- 历史文化底蕴深厚的地区产品数量较多
- 经济发达地区的产品保护意识更强

### 6.3 感官文本挖掘结果

**语言特征**：
- 产品描述中频繁使用形容词，平均每个产品描述包含15-20个形容词
- 不同产品的描述词汇具有明显差异
- 同一产品的描述词汇具有一定共性

**地域特色**：
- 南方产品描述更注重细腻的感官体验，如"明亮"、"清甜"、"油润"等
- 北方产品描述更强调粗犷的质感，如"鲜艳"、"厚实"、"丰满"等
- 不同地区对颜色的偏好存在明显差异

**颜色偏好**：
- 红色是最受欢迎的产品颜色描述，尤其在喜庆食品中
- 绿色在农产品中使用频率最高
- 黄色在粮食类产品中较为常见

## 7. 结论与展望

### 7.1 项目总结

本项目成功完成了中国国家地理标志产品的数据收集、处理和分析任务，主要成果包括：

1. **构建了完整的数据库**：收集了4400+个地理标志产品的详细信息，覆盖31个省级行政区
2. **实现了全流程技术方案**：从PDF下载、信息提取到数据分析的完整技术流程
3. **生成了丰富的分析结果**：揭示了地理标志产品的区域分布特征和感官描述规律
4. **提供了可视化展示**：生成了地理分布热力图、词云图等多种可视化结果

### 7.2 技术创新点

本项目在技术实现上具有以下创新点：

1. **多源数据融合**：整合了原质监总局和国家知识产权局的公告数据
2. **智能信息提取**：使用AI大模型，将复杂的规则信息提取范式转换为生成范式
3. **深度文本分析**：使用HanLP进行多维度的自然语言处理
4. **多维度可视化**：从地理、文本等多个维度展示分析结果

### 7.3 存在的不足

本项目在实施过程中也发现了一些不足：

1. **PDF提取精度**：由于是生成范式，PDF的文本准确率有待提高
3. **分析深度**：对产品特征的分析可以进一步深入
4. **交互性**：可视化结果的交互性可以进一步增强

### 7.4 未来展望

基于本项目的成果，未来可以从以下几个方向进行扩展：

1. **数据持续更新**：建立自动化的数据更新机制，实时追踪新发布的地理标志产品
3. **多维度分析**：结合经济、文化等因素，开展更深入的交叉分析
4. **应用系统开发**：开发地理标志产品查询和分析的Web应用系统
